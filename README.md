# Symfony-based skeleton

## Local development

### Commands

First start:
```shell
make pre-start-local
```

Start / restart:
```shell
make start-local
```
Application root: http://localhost:8001/

Stop:
```shell
make stop-local
```

Run tests:
```shell
make test-local
```
See [coverage](coverage/index.html) directory for coverage HTML report

Check code:
```shell
make check-local
```

Fix code:
```shell
make fix-local
```

Enable Git hooks:
```shell
make enable-git-hooks
```
